Modulos de entrada y salida

Las computadoras disponen de los módulos de entrada/salida para poder efectuar operaciones entre el procesador y un periférico.

Los módulos son interfaces y se utilizan para facilitar las operaciones de entrada y salida entre los periféricos y la memoria o procesador.

Estos módulos se conectan con el procesador y la memoria principal, y cada uno controla uno o mas dispositivos externos

Entrada/Salida programada

Para hacer la operación solicitada, el procesador ejecuta un programa que controla la operación de entrada o salida.

Sincronización:

El programa ejecutado revisa el estado del periférico de manera constante, así se sabe si está preparado para comenzar. Mientras se hace el proceso de sincronización, el procesador se dedica solamente a esta tarea y no puede realizar otros procesos.

Intercambio de dato:

En este proceso si la operación es de entrada, el procesador lee el registro de datos del modulo y lo guarda en memoria; en cambio si es de salida, el procesador toma de la memoria el dato que se enviará al periférico y lo escribe en el registro de datos del modulo.

Entrada/Salida mediante interrupciones

El programa genera una orden de entrada o salida Y se ejecuta hasta que el hardware lo interrumpe para indicar que la operación terminó.

La entrada y salida con interrupciones es más eficiente que la programada pero requiere la intervención activa del procesador para transferir los datos entre la memoria y el modulo.

Este método pretende evitar que el procesador se pare o tenga que hacer trabajo improductivo mientras espera a que el periférico esté listo para la operación, así se aprovecha el tiempo para ejecutar otros programas

ntrada/Salida directa a memoria

Un procesador toma el control de la operación para la transferencia de datos El modulo DMA es capaz de imitar al procesador y de transferir datos desde memoria a través del bus del sistema.
El modulo solo utiliza el bus cuando el procesador no lo necesita.

Esta técnica es eficiente para transferir bloques enteros de datos.
El procesador programa la transferencia y encarga a un nuevo elemento conectado al bus del sistema hacer toda la transferencia
Al acabar, el elemento avisa al procesador y este usa el tiempo que dura la transferencia para hacer otras tareas.
Con esta técnica el procesador no tiene que llevar a cabo toda la sincronización e intercambio de datos.

Canales y procesadores de Entrada/Salida

El canal de entrada y salida es una ampliación del concepto de DMA. Este canal puede ejecutar instrucciones de entrada y salida, por lo que puede controlar las operaciones pertinentes.

Un canal selector controla varios dispositivos de alta velocidad y se dedica a transferir datos a uno de ellos. Es decir el canal de E/S selecciona un dispositivo y hace la transferencia.

Un canal multiplexor puede manejar la entrada y salida de varios dispositivos al mismo tiempo. Para dispositivos lentos, un multiplexor de byte acepta o transmite caracteres tan rápido como es posible a varios dispositivos
